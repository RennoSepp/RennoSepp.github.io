Scoring System

3 Points Exact score predicted

Example: predicted 2-1, actual result 2-1.

2 Points Correct goal difference

Example: predicted 3-1, actual result 2-0.

1 Point Correct winner or draw

Example: predicted 2-0, actual result 1-0.

0 Points Incorrect prediction

Example: predicted home win, actual result away win.

Prediction Deadline

Predictions are automatically locked when a match starts. Any predictions submitted after kickoff are rejected by the system.

About This Project

This World Cup Prediction Game was vibecoded in approximately one hour.

While every effort has been made to prevent issues, unexpected bugs, visual glitches, questionable design decisions, or acts of football-related chaos may still occur.

Fair Play

This game runs on the honor system.

Please only edit your own predictions and respect other participants' entries. Accidental edits happen, but intentionally changing someone else's predictions is a guaranteed way to get roasted in the group chat.

Be competitive, not chaotic. ⚽

Bugs & Emotional Support

For bug reports, feature requests, or emotional support after a bad prediction, email renno.sepp@gmail.com.
